Class 5

Attention

  • Should be adjusted to what a potential user needs.
  • Consider the user knows what they want or if they need suggesting.
  • The information should be present, so it is easy for a user to identify what they need when they find it.

Perception

  • Refers to how information is acquired.
  • Should not make it so that distractions are unintended through things like colors.

Memory

  • Products should be designed for recognition.
  • Use chunking and categorization to aid recall.
  • Provide consistency in things like navigation, menus, options, format.

Learning

  • Design for comprehension, readability, and durability.

Reading

  • Information that may want to be recalled should be in written form.
  • Audio should consider accessibility.
  • Speech recognition can allow for additional accessibility or potential increase in efficiency.

Problem-Solving

  • Design for all types of users expertise.
  • Ethical responsibilities in making sure users are not improperly swayed into decisions.
  • Feedback should be welcomed when designing.
